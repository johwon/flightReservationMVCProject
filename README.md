📌 주요 기능
#### 항공사정보 
데이터 정렬, 수정, 입력 ,삭제 기능
DB와 자바 데이터 연동
삭제시 자동 백업하는 테이블 생성
항공사 이름과 등급을 출력하는 펑션 생성

삭제시 자동 백업하는 테이블 생성

#### 고객정보 
데이터 정렬, 수정, 입력 ,삭제 기능
